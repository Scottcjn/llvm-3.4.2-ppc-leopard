# Contributing to llvm-3.4.2-ppc-leopard

Thank you for your interest in contributing to llvm-3.4.2-ppc-leopard! We welcome contributions from everyone.

## Getting Started

1. **Fork** this repository
2. **Clone** your fork locally
3. Create a **branch** for your changes: `git checkout -b my-feature`
4. Make your changes and **commit** them
5. **Push** to your fork: `git push origin my-feature`
6. Open a **Pull Request** against the main branch

## Pull Request Guidelines

- Keep PRs focused — one feature or fix per PR
- Write a clear description of what your PR does
- Include relevant tests if applicable
- Follow the existing code style

## Code Style

- Follow existing conventions in the repository
- Keep code readable and well-commented
- Use meaningful variable and function names

## Reporting Issues

- Use GitHub Issues to report bugs or suggest features
- Include steps to reproduce for bug reports
- Be respectful and constructive in discussions
